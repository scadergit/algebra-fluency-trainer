# Feature Roadmap

This document defines the desired functionality of the Algebra Fluency Trainer.

The goal is to build the highest quality Algebra practice application available for students, teachers, and parents.

This is a living document.

---

## Project Goals

The application should:

- Build mathematical fluency
- Encourage daily practice
- Require minimal teacher supervision
- Adapt to student ability
- Be enjoyable without being distracting
- Support students from elementary arithmetic through Algebra I

---

## Core Principles

The application should always be:

- Fast
- Keyboard friendly
- Responsive
- Accessible
- Predictable
- Easy to use

The interface should never get in the student's way.

---

## Practice Modes

### Standard Practice ✅

Unlimited practice problems. Immediate feedback. Statistics tracked.

### Timed Practice ✅

Student selects: 30 seconds, 1 minute, 2 minutes, 5 minutes, 10 minutes.

Statistics include problems completed, accuracy, and average response time.

### Target Score

Practice until 20, 50, or 100 correct.

### Target Accuracy

Continue until 90%, 95%, or 100% accuracy.

### Speed Drill

Focus entirely on automaticity. Very fast transitions. Minimal animations.

### Mixed Review

Randomly select problems from multiple enabled skills. Weighted by mastery.

### Adaptive Practice

Increase difficulty automatically. Reduce difficulty after repeated mistakes. Favor weak skills.

---

## Student Feedback

Immediately indicate: Correct / Incorrect / Skipped.

Display: Correct answer, explanation, running statistics.

Future: Hints, multiple explanations, animated solution.

---

## Statistics

### Current Session ✅

- Correct
- Incorrect
- Accuracy
- Current streak
- Best streak
- Average response time (correct answers only)

### Lifetime

- Total problems
- Time practiced
- Mastered skills
- Accuracy history

---

## Skill Progress

Every skill tracks:

- Attempts
- Accuracy
- Speed
- Confidence
- Mastery level

---

## Student Profiles

Support multiple students. Each profile stores settings, progress, statistics, mastery, and daily streak.

---

## Dashboard ✅

Currently shows navigation to Practice, Statistics, and Settings.

Future: Today's practice, weekly/monthly summaries, skills needing attention, recommended next practice.

---

## Practice History ✅

Stores date, duration, skills practiced, accuracy, and speed. Persisted to localStorage.

Future: Filtering, cloud sync.

---

## Gamification

Subtle. Never distracting.

Possible features:

- Daily streak
- Weekly streak
- Practice calendar
- Milestones
- Achievement badges
- Level progression

Avoid excessive animations.

---

## Keyboard Support ✅

- Type answer
- Enter = Check
- Escape = Skip
- Tab navigation
- Focus automatically returns to answer box after every answer

---

## Accessibility

Support:

- Screen readers
- High contrast mode
- Large fonts
- Reduced motion
- Keyboard-only navigation

---

## Themes

Support: Light, Dark, High contrast.

---

## Problem Generation

Problems should never feel repetitive.

Support: Difficulty levels, random variation, multiple strategies, equivalent forms.

## Hints

Hints should never reveal the full answer immediately.

- Hint 1: Small clue
- Hint 2: Larger clue
- Hint 3: Walkthrough

---

## Explanations

Every problem should support explanations. Eventually multiple explanation styles: visual, step-by-step, mental math, teacher style.

---

## Difficulty Levels

Every skill should support: Easy, Medium, Hard, Adaptive, Custom.

---

## Settings ✅

Current: max number, negative numbers, fractions, decimals, enabled skills.

Future: difficulty profiles, adaptive mode, sound, animations, theme, keyboard shortcuts.
